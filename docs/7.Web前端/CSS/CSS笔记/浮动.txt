
1、浮动的元素
与其它浮动元素并排。
宽高默认为文字的宽高。
能设置宽高。若设width:100%，则和父容器一样宽。

2、
通过浮动，可以让两个元素并排，并且设置其宽高(这在标准流里是实现不了的)。浮动的元素宽高默认就是文字的宽高。
浮动的性质：脱标，贴靠，字围(能挡住盒子，但是挡不住盒子里的文字)
浮动就是用来做“元素并排”的。

3、清除浮动：

1、一个div默认不能被浮动的后代标签撑出高度。
2、有高度的盒子，才能关住浮动的子孙。
3、一个浮动的元素，其父亲必须要有足够的高度，这样才会不影响之后的元素(无论后面元素是否浮动)。

4、clear:both，清除别人的浮动对自己的影响。(这种方法会使margin失效???)

4、div加上overflow:hidden之后便可以被浮动的子元素撑出高度，此时margin也可以用(奇淫技巧)。

5、隔墙法：用一个有高度的、透明的div作分割。
.clear{clear:both;height:10px;}

6、内墙法：此方法可以撑出其父亲的高(奇淫技巧)。
